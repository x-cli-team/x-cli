## How to Contribute

Whether you're reporting bugs, suggesting features, or contributing code, every contribution helps make Grok One-Shot better for everyone.

### Quick Start Contributing

1. **Star the repo** on GitHub
2. **Join our Discord** for real-time discussions
3. **Try Grok One-Shot** and share your experience
4. **Report issues** or suggest improvements
5. **Contribute code** via pull requests
