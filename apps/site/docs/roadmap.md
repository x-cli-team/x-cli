# Grok One-Shot Roadmap

Our development roadmap outlines planned features and improvements for Grok One-Shot.

## Current Version: v1.1.116

### Recently Completed

**Core Features:**

- Plan Mode with Shift+Tab twice activation
- MCP (Model Context Protocol) integration
- Multi-file operations with transaction support
- React/Ink terminal UI with professional feedback
- Session management with auto-save
- Comprehensive tool system (25+ tools)

**Documentation:**

- Complete Claude Code documentation sync
- Architecture and community pages
- Installation and configuration guides
- API reference and troubleshooting
